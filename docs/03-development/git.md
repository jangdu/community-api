# Git 사용법

NestJS 프로젝트와 동일하다. IntelliJ GUI 또는 터미널 둘 다 가능.

## IntelliJ GUI

| 동작 | 단축키 (Mac) |
|---|---|
| Commit | **Cmd + K** |
| Push | **Cmd + Shift + K** |
| Pull | **Cmd + T** |
| Git 로그 보기 | 하단 **Git** 탭 |

Cmd + K를 누르면 변경된 파일 목록 + diff를 한눈에 보면서 커밋할 수 있다.

## 터미널

```bash
git add .
git commit -m "메시지"
git push
```

IntelliJ 하단 **Terminal** 탭에서 실행 가능.
